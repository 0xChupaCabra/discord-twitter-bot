# discord-twitter-bot

## How to use

1. Install [`Node.JS`](https://nodejs.org/en/download/current/).

2. Get the Repo
```sh
git clone https://github.com/PrincessAkira/discord-twitter-bot && cd discord-twitter-bot
```

3. Install the bot
```sh
 npm install -g pm2 && npm install
 ```

4. Start the Bot
```sh
npm start
```
